# Gunslinger Powers

### High Jump
Thanks to powder boost, some gunslingers can perform so called High Jump - strong jump up that lets them accelerate
up a lot more - it uses lots of angel power.

### Double jump
After some time, Gunslinger is able to see dust aggregation points and use them as physical support 
for a jump - therefore perform a double jump. It uses small amount of angel power. 

### Dash (+Slash)
#### Level 1
With angel power it is possible to do a quick space shift.

#### Level 4 Dash - Slash
When special power is obtained and angel power is present, Gunslinger can perform Slash movement, which dashes 
the obstacles and simultaneously slashes them.
