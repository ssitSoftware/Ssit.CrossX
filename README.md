CrossX is a retro 2D game engine created for fast and seamless 2D games development.
Engine is composed of given parts:
1. Rendering API
2. UI Framework
3. XXGames Framework including Aabb physics implementation.
4. Editor app module.
5. SDL based reendering API implementation.
